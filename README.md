# Movie Chart

A platform to track how many times you watched a movie and organize by genre.

### Ruby version
```
2.6.4
```

### Rails version
```
6.0.2
```

### Configuration
```
bundle install
yarn install
```

### Database creation
```
bundle exec rails db:setup
```

### How to run the test suite
```
bundle exec rspec
```

### Run the server
```
bundle exec rails server
```

### Check `localhost:3000`
